# 🎓 KET 学习营 — 专业版

> 为9岁四年级小朋友量身打造的 KET 考试备考应用，每天只需20分钟！

## 📱 在线使用

直接用浏览器打开 `index.html` 即可使用，无需安装。支持手机、平板、电脑。

也可以部署到 GitHub Pages、Vercel 等平台在线访问。

## ✨ 功能模块

### 每日20分钟学习计划

| 模块 | 时间 | 内容 |
|------|------|------|
| 📝 单词关卡 | 5分钟 | 每天10个A2高频词，选择+例句+发音 |
| 📖 阅读理解 | 5分钟 | A2级别短文，多难度选择 |
| 👂 听力练习 | 5分钟 | Web Speech API 真实语音朗读 |
| ✏️ 真题训练 | 5分钟 | 按星期分配KET题型 |

### 真题训练安排

| 星期 | 题型 | 题数 |
|------|------|------|
| 周一 | Reading Part 1（短消息理解） | 10题 |
| 周二 | Reading Part 2（匹配题） | 10题 |
| 周三 | Reading Part 3（长文章） | 10题 |
| 周四 | Reading Part 4-5（完形填空+填词） | 10题 |
| 周五 | Writing Part 6（写作练习） | 10题 |
| 周六 | 听力综合模拟 | 随机 |
| 周日 | 全套综合模拟 | 10+题 |

## 📊 数据统计

- 150个 KET A2 高频词汇（15天循环）
- 15篇阅读理解（简单/中等/困难）
- 10段听力素材（真实场景）
- 60+ 道真题（覆盖所有KET题型）
- 12个成就徽章

## 🎨 设计特点

- 蓝色/绿色主色调，活泼适合小朋友
- 响应式设计（手机/平板/电脑）
- 答对动画+音效鼓励，答错温和提示
- 离线可用，纯前端，无需服务器
- localStorage 保存数据，关闭不丢失

## 🛠️ 技术栈

- HTML5 + CSS3 + 原生 JavaScript
- Web Speech API（语音朗读）
- localStorage（数据存储）
- Web Audio API（音效反馈）
